關於我們

LAHG(Live Action Hunting Game) 是一家今年剛成立的新創公司，致力於帶給用戶良好的大逃殺遊戲體驗

一群背景各異的年輕人，在因緣際會下接觸到"魷魚遊戲"這個知名影集，而後突發奇想，打造出這個與眾不同的"大逃殺"遊戲。

LAHG 提供的服務內容

我們提供使用者一款市面上獨創的大逃殺手遊APP

同名手遊 LAHG (Live Action Hunting Game) 是一款創新的大逃殺手遊APP，利用獨特的虛實整合技術，玩家可藉由APP在現實世界中來場驚心動魄的大逃殺，而後配合地方創生計畫取得合法場地並落實企業社會責任。

LAHG 提供兩種角色 : Predator 和 Prey。玩家配對成功後，在限制時間內，Predator 必須找到所有的 Prey，否則 Prey 方獲勝。

Predator 獵人

你是一個在森林中狩獵的獵人，在狩獵過程中，你陸續發現獵物留下的痕跡。請根據線索，找到獵物吧!

Prey 獵物

你是一個在森林中生存的獵物，你發現有個獵人想要狩獵你。仔細觀察獵人的行動，努力逃離獵人的魔掌吧!

為使遊戲能順利的進行並充滿緊張感，我們設有"線索"的提示，讓扮演兩種角色的玩家能有資訊作為遊戲時的判斷參考依據。

Predator 獵人

  • 線索 :
    • 每十分鐘會提示 Prey 十分鐘前附近地標，Predator 也可以利用道具收集更多資訊
  • 同伴 :
    • Predator 在 APP 上可以看見 Predator 的所在位置

Prey 獵物

  • 線索 :
    • 每十分鐘會提示 Predator 十分鐘前附近地標，Prey 也可以利用道具收集更多資訊
  • 同伴 :
    • Prey 在 APP 上可以看見 Prey 的所在位置

扮演兩種角色的玩家只要達成以下各自的條件即可獲得遊戲的勝利!

F.A.Q

常見問題

使用者提供的定位資料一概會在遊戲結束後停止傳送資料，本公司亦不會保留定位資料，敬請放心使用。

使用者可至手機應用商店下載遊戲專用APP並發起遊戲，遊戲另有特殊的貨幣、道具和月卡，可透過小額付費取得，遊玩過程一律保證免付費。

我們會與擁有合法且合宜場地的地方政府、民間借用場地，在舉辦大逃殺活動的同時，本公司亦可協助地方創生，期望能協助地方發揮特色，吸引產業進駐、人口回流，促進地方繁榮並善盡企業社會責任。

提供保險服務係非本公司業務範圍，惟使用者若有需要，本公司可協助進行投保業務，但會收取後續相關費用。因活動場所不是本公司營業登記場所，故投保的類別將會是"旅行平安險"。

秉持著公平遊戲的原則，我們額外提供許多特色道具供使用者購買，期望使用者能活用這些道具玩出一場不一樣的大逃殺。道具的使用或許能讓贏的機會增加，但我們也有對強力道具進行限制，玩家若能活用場地亦可找到規避的辦法。

私人客服助理係"鋼鐵雄獅"等級用戶專屬服務，用戶可在第一時間以私訊的方式與私人客服助理進行遊戲意見的回饋與溝通，惟須注意服務時間同營業時間，以免無人回應。其餘等級的用戶則可由服務電話或電子郵件與一般客服聯絡，在溝通上可能會有所延遲，敬請多多包涵。

Special Item

道具介紹

時間炸彈 (獵物專用)

獵物設下時間炸彈，當獵人無法在時間內到達獵物身邊100公尺內，獵物直接獲勝。


獲得方式:

月卡簽到(極低機率) / 商店購買

猴の香蕉陷阱 (獵人專用)

獵物碰到香蕉時，會強制停留在原地1分鐘，不遵守則強制出局。


獲得方式:

月卡簽到 / 商店購買

不科學磁鐵 (獵物專用)

與其他獵物共享道具功能，但只能共享4星以下道具。


獲得方式:

月卡簽到 / 商店購買

YO! 捕獸夾 (獵人專用)

設下捕獸夾，當獵物接近捕獸夾範圍內100公尺，則會通知獵人目前獵物的位置。


獲得方式:

月卡簽到 / 商店購買

戳仔牌釘子 (獵物專用)

透露除自己以外周圍距離100公尺內所有其他獵物的位置給獵人，效果持續5秒鐘。


獲得方式:

月卡簽到 / 商店購買
